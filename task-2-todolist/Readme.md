# To-Do List Application (Console Based)

This is a simple Python CLI To-Do app with persistent storage using a text file.

## Features
- Add tasks
- Remove tasks
- View tasks
- Automatic saving to tasks.txt
- File handling, lists, string methods

## Run the Project
python todo.py

## Files
- todo.py
- tasks.txt (auto created)

## Concepts Used
- Lists
- File handling
- Context managers
- strip()
- append(), pop()
